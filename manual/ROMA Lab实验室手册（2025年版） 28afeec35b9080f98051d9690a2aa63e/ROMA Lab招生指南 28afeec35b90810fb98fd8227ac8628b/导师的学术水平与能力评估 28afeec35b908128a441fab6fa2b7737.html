📚 导师学术水平与能力评估

👤 个人简介

孙相国博士现任东南大学计算机科学与工程学院（计软智学院）青年首席教授（Full Professor），博士生导师。他的研究方向聚焦于数据挖掘、在线社交网络与社会计算，并致力于探索面向社会网络的下一代通用人工智能，其核心攻关方向为非线性数据通用智能模型。

🏆 学术成就与影响力

孙相国博士在学术研究上取得了显著的成就，其成果在国际上获得了广泛认可：

🥇 顶级学术荣誉

  • KDD 2023 最佳研究论文奖：以第一作者身份荣获数据挖掘顶会KDD 2023"最佳研究论文奖"（全会仅评选1篇）。该会议被公认为计算机领域难度最高的国际顶尖会议之一，而此次获奖是该会议创立三十年以来，中国大陆及港澳地区学者首次获此殊荣。历史上以第一作者获此荣誉的学者还包括斯坦福大学Jure Leskovec教授、推荐系统奠基性学者Yehuda Koren、华盛顿大学Pedro Domingos教授等。
  • 鉴于其在社会计算与社交网络方面的研究，他被中国人工智能学会（CAAI）社会计算专委会评选为“2023年度社会计算青年学者新星”，面向全球华人青年学者评选，同年度仅10人

📊 高影响力论文

  • 最具影响力论文：多篇论文入选KDD、IJCAI等顶会的"最具影响力论文"榜单
  • 高被引论文：多篇论文进入KDD, WSDM, CIKM, IJCAI等会议Web of Science高被引前1%及前10%
  • 学术影响力：截至2025年10月，Google Scholar引用达2160次，H-index为24

🔬 理论成果落地

他的研究兼具理论深度与实践价值：

  • 异质超图模型：所提出的异质超图模型被知名开源图计算库EasyGraph采纳为标准算法
  • 通用图提示范式：开创的"通用图提示"范式，在蛋白质结构预测中效果超越业界标杆AlphaFold-Multimer，展现出强大的跨领域潜力
  • 开源工具包ProG：主导开发的开源工具包ProG已成为该领域广泛采用的评测标准，获得GitHub Stars 560+

💼 技术应用与社会服务

孙相国博士积极推动研究成果的转化与应用，并积极参与学术服务：

  • 技术应用实践：
    • 其核心算法作为智慧公安系统的核心组件，落地于南京及江苏省公安刑侦部门，用于大数据情报研判。
    • 在疫情期间，相关模型支撑了企业的网格化疫情分析系统，取得了显著的经济效益。
  • 学术服务：他于2023年被中国人工智能学会(CAAI)社会计算与社会智能专业委员会遴选进入专委会，是当时专委会一百多位委员中唯一一个博士后。现担任专委会副秘书长和专委会第三届新星执委会主席。
  • 此外他还担任 Information Processing & Management、Big Data Mining and Analytics 等国际知名期刊的编委，并长期担任SIGKDD、NeurIPS、ICML等顶级会议的程序委员会委员或评审专家，并多次获评KDD等顶尖会议的杰出评审专家。

📝 代表性学术成果

近年来以第一作者和通讯作者（指导学生的工作）在SIGKDD, VLDB, TKDE, NeurIPS, ICML, ICLR, IJCAI, WWW, SIGIR等顶尖会议期刊上发表论文40余篇. 下面列举的主要是我过去指导的学生一作工作，以及一小部分我一作的工作。关于更多我指导的学生一作的论文、我为一作的论文，以及我的代表作论文，请访问我的个人学术主页获取更多信息。

  • [KDD 2023, CORE A*, CCF A] All in One: Multi-task Prompting for Graph Neural Networks.
    这篇在ChatGPT爆发前夜诞生的论文，不仅在学界为通用非线性数据智能开辟出全新的路径，捧回了KDD三十多年历史上首个大陆与港澳的最佳论文奖，更在生活里默默发力，让我在相亲的时候变得稍微自信了一些，这间接使我遇到了现在的妻子——虽然时至今日，她对我的研究仍一窍不通，但这丝毫不影响她是我此生最成功的“引用”。
    Best Paper Award, ranked as Top 3 Most Influential Papers by Paper Digest 2025-03, and TOP 1% Most Cited Papers by Web of Science 2025-06

  • [ICML 2025, CORE A*, CCF A] Does Graph Prompt Work? A Data Operation Perspective with Theoretical Analysis.
    这篇硬核论文只有三位作者，一作是我指导的大一本科生，Mr. Qunzhong Wang, 来自香港中文大学，我是这篇论文的唯一通讯和共一第二。这篇工作做完时，Qunzhong已经是大二了。我记得他第一次找我的时候，看着这个大一的小孩子，我其实是有一点顾虑的，担心他能否有韧性，便试探性地给了他几个非常简单的idea，也给了个非常容易出论文的idea，但让我吃惊的是，他直接拒绝了这些idea，他希望能做出一篇别人难以做出来的东西。为此，我不得不把我长期做的这个艰难的问题分享给他，提供了主要的思路和技术，让我开心的是，这是一个从一开始就具有卓越taste的本科生，凭借他扎实的数学功底，我们终于一步一步啃下了这块硬骨头，这也成为目前为止我最满意的一篇工作，虽然现在不怎么被人关注，但我相信十年之后人们会发现它的价值。

  • [NeurIPS 2024, CORE A*, CCF A] ProG: A Graph Prompt Learning Benchmark.
    一作是我协助指导的硕士研究生，我为通讯。目前已在香港科技大学（广州）都博。在这篇工作中，我独立领导研究生团队开发通用图提示工具箱ProG获得Github Stars 550+, 成为图提示方法的标准工具，被上海交大等多项同行研究使用，成为他们实验分析的重要工具。

  • [ICLR 2025, CORE A*, spotlight ac 5.1%] Graph Sparsification via Mixture of Graphs.
    一作是我指导的本科生，来自同济大学，我为共一。2个月完成研究工作，目前已在新加坡国立大学读博

  • [TKDE 2023, CORE A*, CCF A] Self-supervised Hypergraph Representation Learning for Sociological Analysis.
    论文研究正值疫情管控、博士毕业、工作尚未落实这三段奇幻交织的人生诡异时刻。那段在破旧出租屋里的岁月，疫情如浪，我于失业与捉襟见肘的交织窘迫中，在冒烟的键盘前写完了这篇自监督超图的论文初稿。它诞生于咳嗽与焦虑，失业与入不敷出中，也见证了师友的慷慨与扶持——在梦碎声四起的日子里，是这些温暖托住了我。愿所有赤子之心，终能得偿所愿。

  • [TKDE 2024, CORE A*, CCF A] Counter-Empirical Attacking based on Adversarial Reinforcement Learning for Time-Relevant Scoring System.
    在微软亚洲研究院，从一窍不通到论文完稿，全心投入，历时半年，从灵感到实验，每一行代码皆亲手铸就，它如初春北京未及道别的燕子，小众而真挚，虽暂未被广泛引用，却是我心中笃信的好作品。北京不再是我的过路地，因为我曾经，面朝海淀南路的黄昏，骑着小黄车，追逐过夕阳。

  • [TheWebConf 2021, CORE A*, CCF A] Multi-level Hyperedge Distillation for Social Linking Prediction on Sparsely Observed Networks.
    孙老师学生时代的第一篇CCF A，课题组零的突破，从此学生时代收获了一堆实验室迷妹迷弟

  • [TKDE 2024, CORE A*, CCF A] Attention Is Not the Only Choice: Counterfactual Reasoning for Path-Based Explainable Recommendation.
    一作是我长期协助指导的博士研究生，来自悉尼科技大学，我为通讯。她因为论文录用过于兴奋，扭折了脚踝，拄了一个多月的拐，由此可见这篇论文的含“金”量还是很高的，该生目前已在重点大学任职

  • [SIGIR 2025, CORE A*, CCF A] Adaptive Coordinators and Prompts on Heterogeneous Graphs for Cross-Domain Recommendations.
    一作是我协助指导的博士研究生，来自CUHK，我为通讯

  • [KDD 2024, CORE A*, CCF A] All in One and One for All: A Simple yet Effective Method towards Cross-domain Graph Pretraining.
    一作是我协助指导的博士研究生，来自香港科技大学（广州），我为通讯，ranked as the Most Influential Paper in KDD24 by Paper Digest 2025-03

  • [CIKM 2024, CORE A, CCF B] When LLM Meets Hypergraph: A Sociological Analysis on Personality via Online Social Networks.
    一作是我独立指导的硕士研究生，来自UC Berkeley，我为共一和唯一通讯。该生非科班（艺术系），0基础3个月完成论文研究，目前已在美国知名大学攻读IT类博士。
